# URL Shortener

A full-stack URL shortener with user authentication and click analytics built using Node.js, Express.js, MongoDB, and EJS.

## Features

- User signup and login
- Cookie-based authentication
- Protected routes using custom Express middleware
- Automatic short URL generation
- Dynamic redirection to original URLs
- User-specific URL management
- Click analytics with timestamped visit history
- Server-side rendered UI using EJS

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- JavaScript
